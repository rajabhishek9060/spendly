## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/amritrajrajput/spendly.git
cd spendly
```

### 2️⃣ Create virtual environment

```bash
python -m venv cloudevenv
```

### 3️⃣ Activate environment

```bash
# Windows
cloudevenv\Scripts\activate

# Mac/Linux
source cloudevenv/bin/activate
```

### 4️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 5️⃣ Run the application

```bash
python main.py
```
